Docker容器时间不匹配

1. 问题描述：

执行命令

docker cp /etc/localtime  zoo2:/etc/

报错

Error response from daemon: Error processing tar file(exit status 1): invalid symlink "/etc/localtime" -> "../usr/share/zoneinfo/Asia/Shanghai"
2. 解决办法(一)：

第一步：变换命令

docker cp /usr/share/zoneinfo/Asia/Shanghai zoo2:/etc/localtime

第二步：重启容器

docker restart  zoo2

第三步：验证是否修改成功

docker exec -it  zoo2 /bin/bash

回车后输入date即可

3. 解决办法（二）：

build 镜像添加命令

FROM java:8
MAINTAINER Caixin
ADD square-root.jar square-root.jar
EXPOSE 8080
RUN /bin/cp /usr/share/zoneinfo/Asia/Shanghai /etc/localtime && echo 'Asia/Shanghai' >/etc/timezone
ENTRYPOINT ["java","-Djava.security.egd=file:/dev/./urandom","-jar","/square-root.jar"]
